# Data Manipulation

From the terminal...

- html: ``` yo reveal:slide "Slide Title" ```
- markdown: ``` yo reveal:slide "Slide Title" --markdown ```
- add attrs: ``` yo reveal:slide "Slide Title" --attributes ```
- add notes: ``` yo reveal:slide "Slide Title" --notes ```
